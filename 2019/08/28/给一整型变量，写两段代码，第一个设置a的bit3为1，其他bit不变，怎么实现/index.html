给一整型变量a，写两段代码，第一个设置a的bit3，第二个清除a的bit3.“a的bit3”

八月 28, 2019

|操作,当操作数为1时,结果为1,当操作数为0时,结果不变.
&操作,当操作数为1时,结果不变,当操作数为0时,结果为0

这个a是一个整形的,占8位,所以bit3应该是第三位。因为是从第零位开始计算的。

因此：设置a的bit3为1的方法就是将a和00001000作|操作,这样其他位都不变,只有第3位变成1。
清除a的bit3的方法就是将a和11110111作&操作,这样第3位变成0,其他位不变。

所以最终结果应该是：

#define BIT3 (0x1<<3) 
static int a; 
void set_bit3(void) 
{ 
    a |= BIT3; 
} 
void clear_bit3(void) 
{ 
    a &= ~BIT3; 
}

bit0 = 0x1<<0 = 0001
bit1 = 0x1<<1 = 0010
bit2 = 0x1<<2 = 0100
bit3 = 0x1<<3 = 1000
